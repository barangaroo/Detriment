Detriment is a WiFi network security scanner. To test:

1. Open the app and complete onboarding
2. Grant location permission (required for WiFi name detection)
3. Tap the SCAN button to scan your network
4. The app will discover devices on your local network via TCP probing and display results with a security score

Note: The app requires an active WiFi connection to function. It uses local network access to discover devices via TCP connection attempts and reads the ARP cache for MAC addresses. Lookup and optional Improve Detection requests send only MAC prefixes and open port numbers to our API, not full MAC addresses, local IP addresses, device names, network names, or account identifiers.

If testing on simulator, the app uses mock data to demonstrate all features.
